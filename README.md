# Título do Projeto 
Geração de projeto de subsistema de visão computacional de engenharia reversa de PCBs com pipeline modular

## 📒 Descrição
pré-processamento com OpenCV;

detecção de componentes;

segmentação instance-level;

classificação fina;

OCR de serigrafia/referência;

estimação de package/footprint;

resolução por contexto;

revisão humana;

exportação JSON.

## 🤖 Tecnologias Utilizadas
perplexity.ai

## 🧐 Processo de Criação
Utilizando engenharia de prompt , a IA criou instruções detalhadas para a construção detalhada e etapas necessárias para alcançar o objetivo de engenharia reversa de pcbs

## 🚀 Resultados
A definição detalhada de um projeto para a construção de um sistema python completo para a detecção, a partir de uma foto, dos componentes eletrônicos em uma placa eletrônica
